# PhosphorVR

VR-тренажёр для обучения операторов управлению электротермической установкой
(руднотермическая печь **РКЗ-48**) производства жёлтого фосфора. Выпускная
квалификационная работа.

Обучаемый в VR-очках управляет печью с виртуального пульта, а инструктор задаёт
сценарии обучения и анализирует протоколы на ПК. Математическая модель печи —
детерминированная (чистый C#), сверена с эталонным WinForms-расчётом.

## Состав проекта

| Часть | Где | Назначение |
|---|---|---|
| **VR-клиент** | `Assets/` (Unity) | приложение обучаемого: вход, выбор сценария, пульт, графики, мнемосхема |
| **Сервер-инструктор** | `InstructorServer/` | готовая сборка: TCP-сервер + интерфейс инструктора/админа + база SQLite (`db.db`) |

Исходный код сервера-инструктора (`PhosphorManager`) ведётся отдельно; в репозитории
лежит только его **готовая сборка** для запуска без компиляции.

## Возможности

- **Обучаемый (VR):** вход по логину, выбор сценария, управление подачей сырья и
  электродом с пульта, графики и мнемосхема в реальном времени, обратный отсчёт
  времени обучения. По завершении формируется протокол и отправляется инструктору.
- **Инструктор (ПК):** настройка сценариев (оборудование, сырьё, технологический
  режим с регламентами), просмотр протоколов обучения с данными моделирования.
- **Сценарии и параметры** загружаются из БД (схема EAV), а не захардкожены.

## Запуск

### Сервер-инструктор
1. Требуется установленный **.NET 8 Runtime** (Windows).
2. Запустить `InstructorServer/Phosphor.exe`.
3. Войти как инструктор/администратор; для VR-клиента открыть вкладку сервера и
   указать **реальный LAN-IP** компьютера (порт 5000).

Тестовые учётные записи: `operator/operator` (обучаемый), `instructor/instructor`,
`admin/admin`.

### VR-клиент
1. Открыть проект в **Unity 6** (6000.3.x) с XR Interaction Toolkit и OpenXR.
2. Сцена — `MainScene`. Для проверки на ПК есть XR Device Simulator.
3. Сборка под **Meta Quest** (Android, IL2CPP/ARM64) → установить APK на очки.
4. На экране входа указать IP сервера-инструктора и войти учётной записью обучаемого.
